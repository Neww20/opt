This is a Python program to implement Kruskal’s Algorithm for finding the Minimum Spanning Tree (MST) of a graph. 

A Minimum Spanning Tree connects all vertices with: 

minimum total edge cost  

no cycles  

Shape 

Step-by-Step Explanation 

1. Number of Vertices 

V = 4 

The graph has 4 vertices: 

0  

1  

2  

3  

Shape 

2. Edge List 

edges = [ 
    (0, 1, 1), 
    (0, 2, 3), 
    (1, 2, 2), 
    (1, 3, 4), 
    (2, 3, 5) 
] 

Each tuple represents: 

(u, v, weight) 

Meaning: 

edge between u and v  

with given weight  

Example: 

(0, 1, 1) 

means: 

edge between vertex 0 and 1  

weight = 1  

Shape 

Graph Representation 

Shape 

3. Sort Edges by Weight 

edges.sort(key=lambda x: x[2]) 

This sorts edges using the weight (x[2]). 

Sorted edges become: 

[ 
(0,1,1), 
(1,2,2), 
(0,2,3), 
(1,3,4), 
(2,3,5) 
] 

Kruskal always picks the smallest edge first. 

Shape 

4. Parent Array 

parent = [i for i in range(V)] 

Creates: 

parent = [0,1,2,3] 

Initially every node is its own parent. 

Used for Union-Find to detect cycles. 

5. Find Function 

def find(i): 
    while parent[i] != i: 
        i = parent[i] 
    return i 

Purpose: 

finds the root parent of a vertex  

Example: 

If: 

parent = [1,2,2,3] 

Then: 

find(0) 

goes: 

0 → 1  

1 → 2  

returns: 

2 

 

6. Union Function 

def union(x, y): 
    parent[x] = y 

Purpose: 

joins two sets  

Example: 

union(0,1) 

makes: 

parent[0] = 1 

meaning: 

vertex 0 is connected to set of 1  

 

7. Empty MST List 

mst = [] 
total_cost = 0 

mst stores MST edges  

total_cost stores total minimum cost  

 

8. Main Loop 

for u, v, w in edges: 

Checks every edge one by one. 

 

9. Find Parents 

x = find(u) 
y = find(v) 

Finds root parents of vertices. 

 

10. Cycle Check 

if x != y: 

If parents are different: 

adding edge will NOT create cycle  

so include edge in MST  

If same: 

cycle will form  

skip edge  

 

11. Add Edge to MST 

mst.append((u, v, w)) 

Stores edge in MST. 

 

12. Add Weight 

total_cost += w 

Adds edge weight to total cost. 

 

13. Union Operation 

union(x, y) 

Connects both sets. 

 

Step-by-Step Execution 

Sorted edges: 

(0,1,1) 
(1,2,2) 
(0,2,3) 
(1,3,4) 
(2,3,5) 

 

Edge 1: (0,1,1) 

Different parents: 

add to MST  

MST: 

[(0,1,1)] 

Cost: 

1 

 

Edge 2: (1,2,2) 

Different parents: 

add to MST  

MST: 

[(0,1,1), (1,2,2)] 

Cost: 

3 

 

Edge 3: (0,2,3) 

Both already connected. 

Adding it creates cycle. 

SKIP. 

 

Edge 4: (1,3,4) 

Different parents: 

add to MST  

MST: 

[(0,1,1), (1,2,2), (1,3,4)] 

Cost: 

7 

Now MST has: 

V-1 edges  

3 edges for 4 vertices  

Done. 

 

Final Output
