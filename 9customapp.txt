Step 5: Create Relationship in Salesforce

Relationships are used to connect one object with another object in Salesforce.

Example:

Student Object connected with Course Object
One Course can have many Students

This helps in:

Sharing related information
Managing connected records
Improving database organization
Types of Relationships in Salesforce
1. Lookup Relationship

A Lookup Relationship creates a loose connection between two objects.

Features
Parent and child records are independent
Child record can exist without parent
Deleting parent does not delete child automatically
Example
Student object linked to Course object
Even if Course is deleted, Student record still exists
How to Create Lookup Relationship
Steps
Go to Setup
Open Object Manager
Select Student Object
Click Fields & Relationships
Click New
Select Lookup Relationship
Click Next
Choose Related Object:
Course
Enter Field Label:
Course Name
Click Save
